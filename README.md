FinBoard

Customizable Finance Dashboard built using Next.js.

## Features
- Customizable dashboard with widgets
- Financial API integration (Finnhub)
- Automatic data refresh (polling)
- Persistent dashboard state (localStorage)
- Light/Dark theme support

## Tech Stack
- Next.js
- Financial API: Finnhub (put in .env)
- Charting: custom lightweight components (no external chart library)
- Browser Storage: localStorage

## How to Run Locally
1. Install dependencies

   npm install

2. Run development server

   npm run dev

Create a .env with API key:

NEXT_PUBLIC_FINNHUB_API_KEY=my_api_key
